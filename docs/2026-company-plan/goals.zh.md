## 核心目標

### 1. 營收目標

| 項目 | 目標 |
|------|------|
| 目標營收 | 57.8 億 VND / ~$220,000 USD / ~700 萬 TWD |
| 對比基準 | 2025 年總成本的 2 倍 |
| 損益平衡 | 2027 年 Q2 |

**營收來源：**

| 來源 | 說明 |
|------|------|
| Siraya 服務費 | 薪資 + 10%（現有） |
| 防劫持產品抽成 | 銷售額 5-10%（目標 50 客戶） |
| 維護費 | 5-10 萬 TWD/月 |

**防劫持產品營收潛力（抽成模式）：**

| 項目 | 金額 |
|------|------|
| Siraya 客戶收費 | $5,000-10,000 USD/月/客戶 |
| 目標客戶數 | 50 位 |
| Siraya 月收 | $250,000-500,000 USD |
| Jarvis 抽成 5-10% | $12,500-50,000 USD/月 / 40-160 萬 TWD/月 |

### 2. 組織目標

| 項目 | 目標 |
|------|------|
| 目前人數 | ~8 人 |
| 目標人數 | 15 人 |
| 團隊組成 | 開發 + 專案管理 + 行政 |
| 關鍵職位 | 技術主管（創業家思維） |

**技術主管關鍵能力：**
- 專業實戰：深厚技術底蘊與實際操作能力
- 領導整合：優秀領導力與跨部門資源整合
- 前瞻思維：洞察未來趨勢並解決核心問題

### 3. 產品目標

| 產品 | 目標 |
|------|------|
| 防劫持產品 | 持續優化、開發新功能、產生營收 |
| 客戶後台系統 | 持續開發功能 |

### 4. 國際化目標

- 強化國際化跨國合作
- 整合集團資源
- 頻繁人員交流
- 落實國際化接軌

### 5. 制度目標

- 建立標準化軟體開發流程（SOP）
- 導入明確專案管理制度
- 確保交付品質與時程可控
