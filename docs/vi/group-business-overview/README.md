# Tổng Quan Kinh Doanh Tập Đoàn

## Mô Tả Tài Liệu

Thư mục này chứa cấu trúc kinh doanh tổng thể của tập đoàn, giới thiệu công ty con, và giải thích mối quan hệ tổ chức.

```
Công Ty TNHH Đầu Tư Quốc Tế Unicorn (Công Ty Mẹ)
    │
    ├── Siraya (Nhà cung cấp Cloud/CDN/Bảo mật)
    │
    ├── Jarvis (Phát triển Phần mềm) ← Bạn ở đây
```
