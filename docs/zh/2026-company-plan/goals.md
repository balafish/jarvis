## 核心目標

### 1. 營收目標

| 項目 | 目標 |
|------|------|
| 目標營收 | 57.8 億 VND / ~$220,000 USD / ~700 萬 TWD |
| 對比基準 | 2025 年總成本的 2 倍 |
| 損益平衡 | 2027 年 Q2 |

**營收來源：**

| 來源 | 說明 |
|------|------|
| Siraya 服務費 | 薪資 + 10%（現有） |
| 防劫持產品抽成 | 銷售額 5-10%（目標 50 客戶） |
| 維護費 | 5-10 萬 TWD/月 |

**防劫持產品營收潛力（抽成模式）：**

| 項目 | 金額 |
|------|------|
| Siraya 客戶收費 | $5,000-10,000 USD/月/客戶 |
| 目標客戶數 | 50 位 |
| Siraya 月收 | $250,000-500,000 USD |
| Jarvis 抽成 5-10% | $12,500-50,000 USD/月 / 40-160 萬 TWD/月 |

### 2. 組織目標

| 項目 | 目標 |
|------|------|
| 目前人數 | ~8 人 |
| 目標人數 | 15 人 |
| 團隊組成 | 開發 + 專案管理 + 行政 |
| 關鍵職位 | 技術主管（創業家思維） |

**技術主管關鍵能力：**
- 專業實戰：深厚技術底蘊與實際操作能力
- 領導整合：優秀領導力與跨部門資源整合
- 前瞻思維：洞察未來趨勢並解決核心問題

### 3. 產品目標

| 產品 | 目標 |
|------|------|
| 防劫持產品 | 持續優化、開發新功能、產生營收 |
| 客戶後台系統 | 持續開發功能 |

### 4. 技術目標

#### 4.1 標準化開發流程（Q1）
- Git Flow / Code Review 規範
- CI/CD 自動化部署
- 文件化標準（API Doc, README）

#### 4.2 專案管理制度（Q1）
- Sprint Planning + Retrospective
- 工時估算與追蹤機制
- Issue Tracking 標準化

#### 4.3 Vibe Coding 能力訓練（Q1 啟動，Q2 持續）
- AI 輔助開發工具導入（Cursor / Copilot）
- Prompt Engineering 基礎培訓
- AI Code Review 流程
- 目標：減少重複編碼，提升產出效率

#### 4.4 品質管控（Q2）
- 自動化測試導入
- 單元測試覆蓋率目標
- 上線前 Checklist

### 5. 國際化目標

#### 5.1 跨國專案合作
- 與 **3 間不同集團公司** 啟動 **3 項新專案合作**
- Q2：1 項、Q3：2 項
- 潛在對象：Tomo、勝海、天網、艾瑞爾、宇洋、凱尹
- 排除現有進行中專案（Siraya）

#### 5.2 跨國溝通能力
- 對象：PM、技術主管、行政主管
- 方式：
  - AI 翻譯工具應用（即時溝通）
  - 語言能力培訓
- 目標：能獨立與集團各公司對接
- 全年持續（線上進行）

#### 5.3 跨國交流活動
- 至少 **3 次** 非越南本地交流
- Q1 末：第 1 次
- Q3：第 2 次
- Q4 尾牙前：第 3 次（年會慶功 Party）
