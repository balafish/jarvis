## Q1：Jarvis 與 Siraya 的定位差異

**問題：**
Jarvis 被定位為 Siraya 的 Tech hub，但 Siraya 也在建立自己的 Tech team。如果 Siraya 有自己的 Tech team，Jarvis 的角色是什麼？

**回答：**

Jarvis 一直都是**獨立的越南研發公司**，擁有集團資源共享的優勢，並能直接與國際市場接軌。

| | Jarvis | Siraya |
|---|--------|--------|
| 定位 | 獨立研發公司 | 商務/銷售代理公司 |
| 核心 | 純粹 IT 研發 | 專注市場拓展 |
| 服務對象 | 集團多間公司 | Siraya Tech team 僅服務 Siraya |
| 國際接軌 | 有 | 專屬內部研發中心 |

**合作背景：**

Jarvis 與 Siraya 屬於同集團，一開始各自資源匱乏，因此互相補足。Siraya 2026 年正在籌劃自己的 Tech team，但該團隊是附屬在 Siraya 下的專屬研發中心，不會接觸 Siraya 之外的研發需求。

**未來模式：**

當 Siraya 有自己的團隊後，Jarvis 與 Siraya 合作的現有專案將轉為**外包維護模式**，或在未來某時間點**轉移給 Siraya 研發團隊**。

## Q5：人才流動政策

**問題：**
如果 Jarvis 員工想轉到 Siraya，公司的立場與處理方式是什麼？

**回答：**

每個員工都可以有自己的選擇。對 Ryan 來說，Jarvis 與 Siraya 都是他的公司，彼此不存在競爭問題。

更應該問的是：**你想待在哪間公司？你想達成什麼樣的目標？**

### 重要說明

這兩間公司**並不是提供選擇給員工的人才流通管道**。

如果一個 Jarvis 員工想轉移到 Siraya，就像是：
- **離職 Jarvis** → **被 Siraya 錄取**

反之也一樣。這是用人市場的機制，並不是內部調動。

### 流程

1. **提早一個月告知**
2. **明確交接計劃，完成交接**

### 重要提醒

- 兩邊公司文化與考核方式都不相同
- **結果需要自己負責**

## Q7：工作環境與管理方式差異

**問題：**
Jarvis 與 Siraya 在工作環境、管理方式、工作模式上有什麼差異？

**回答：**

Jarvis 是新創公司，Siraya 是一間已經快五年的公司，兩者的制度與管理模式截然不同。

Siraya 是商務公司，完全是商務與市場導向，沒有研發資源可以共享，但有很好的市場資源，可以結合產品。

| 面向 | Jarvis | Siraya |
|------|--------|--------|
| 公司階段 | 新創 | 成熟（近5年） |
| 導向 | 技術/研發導向 | 商務/市場導向 |
| 管理風格 | 彈性、扁平 | 制度化、流程化 |
| 資源優勢 | 研發能力 | 市場資源 |
