## 資金關係

### Jarvis 資金結構

```
勝海科技 (投資方)
    │
    │ 投資 + 虧損補貼
    ▼
Jarvis (軟體開發)
    │
    │ 提供開發服務
    ▼
Siraya (雲端服務商)
    │
    │ 支付薪資 + 10% 服務費
    ▼
Jarvis 營收
```

**說明：** 
- Jarvis 營收主要來自 Siraya 的服務費
- 虧損部分由勝海科技（投資方）全額吸收
- 目標 2027 Q2 達成損益平衡

## 業務關係說明

### Jarvis 與 Siraya 關係

```
Siraya (雲端/CDN/安全服務商)
    │
    │ 委託開發 + 支付服務費
    ▼
Jarvis (軟體開發)
    │
    ├── 客戶後台管理系統
    │   └── 整合雲服務、CDN、數據監控、帳號管理
    │
    └── 防劫持產品
        └── Anti-Hijacking Solution（2025年末上線）
```

**說明：** Jarvis 目前的主要業務是為 Siraya 開發產品，是 Siraya 的核心研發團隊。

---

### Jarvis 與勝海科技關係

```
勝海科技 (iGaming 研發)
    │
    ├── 投資方：全額吸收 Jarvis 虧損
    │
    └── 過去合作
        ├── Telegram 原生產品開發
        ├── ClickHouse 大數據資料調研
        └── 支付渠道串接
```

**說明：** 勝海科技是 Jarvis 的投資方，Jarvis 剛成立時曾協助勝海科技開發產品。

---

### Jarvis 與天網數位關係

```
天網數位 (技術解決方案)
    │
    │ 目前合作
    ▼
Jarvis (軟體開發)
    │
    └── ClickHouse 大數據服務
        └── 應用於 GC 專案（三方遊戲整合）
```

**說明：** Jarvis 的 ClickHouse 大數據服務應用於天網數位的 GC 專案。

---

### Jarvis 與 Tomo 關係（未來規劃）

```
Tomo (AI 新創)
    │
    │ 未來合作規劃
    ▼
Jarvis (軟體開發)
    │
    └── AI HR 越南市場推行
        ├── 利用 Jarvis 越南在地優勢
        └── 客製化研發合作
```

**說明：** Tomo 曾向 Jarvis 展示 AI HR 產品，未來可能合作推行至越南市場。
