# 公司文件庫 | Company Documents | Tài Liệu Công Ty

此倉庫包含公司重要文件與資料，支援三種語言版本。
This repository contains important company documents, supporting three language versions.
Kho lưu trữ này chứa các tài liệu quan trọng của công ty, hỗ trợ ba phiên bản ngôn ngữ.

## 語言版本 | Language Versions | Phiên Bản Ngôn Ngữ

| 語言 | Language | Ngôn ngữ | 檔案後綴 |
|------|----------|----------|----------|
| 中文 | Chinese | Tiếng Trung | `.zh.md` |
| 英文 | English | Tiếng Anh | `.en.md` |
| 越南文 | Vietnamese | Tiếng Việt | `.vi.md` |

## 目錄結構 | Directory Structure | Cấu Trúc Thư Mục

```
docs/
├── 2026-company-plan/           # 2026 公司計劃
│   ├── README.{zh,en,vi}.md     # 計劃總覽
│   ├── goals.{zh,en,vi}.md      # 年度目標
│   ├── strategy.{zh,en,vi}.md   # 策略規劃
│   ├── budget.{zh,en,vi}.md     # 預算規劃
│   └── timeline.{zh,en,vi}.md   # 時程表
│
├── group-business-overview/     # 集團業務內容與關係說明
│   ├── README.{zh,en,vi}.md     # 總覽
│   ├── subsidiaries.{zh,en,vi}.md   # 子公司介紹
│   ├── business-units.{zh,en,vi}.md # 業務單位說明
│   └── org-relations.{zh,en,vi}.md  # 組織關係圖
│
└── discussion-notes.md          # 討論筆記（內部使用）
```

## 使用方式 | Usage | Cách Sử Dụng

1. 所有文件使用 Markdown 格式撰寫
2. 使用 Git 進行版本控制
3. 可使用 Pandoc 轉換為 PDF/DOCX/PPTX 格式
4. 每個文件都有中文、英文、越南文三個版本
