# 公司文件庫

此倉庫包含公司重要文件與資料。

## 目錄結構

```
docs/
├── 2026-company-plan/      # 2026 公司計劃
│   ├── README.md           # 計劃總覽
│   ├── goals.md            # 年度目標
│   ├── strategy.md         # 策略規劃
│   ├── budget.md           # 預算規劃
│   └── timeline.md         # 時程表
│
└── group-business-overview/ # 集團業務內容與關係說明
    ├── README.md            # 總覽
    ├── subsidiaries.md      # 子公司介紹
    ├── business-units.md    # 業務單位說明
    └── org-relations.md     # 組織關係圖
```

## 使用方式

1. 所有文件使用 Markdown 格式撰寫
2. 使用 Git 進行版本控制
3. 可使用 Pandoc 轉換為 PDF/DOCX/PPTX 格式
